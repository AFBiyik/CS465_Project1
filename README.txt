To Execute:
    Run Executable/Village.html file

You can change scale of entities, max & min river width and max number of entities.
You can select an attractor and place it in the canvas area by clicking.
You can generate a village after at least one attractor is placed.
You can click debug option to see attractor positions (red: house, yellow: tree, blue: rock) and disks.
You can reset village to default state.
You can save village as json file.
You can load village from json file.
